Sobre SecureCheck

SecureCheck es una herramienta que te permite verificar si tu información personal ha sido comprometida en filtraciones de datos conocidas.

Utilizamos la API de Have I Been Pwned (HIBP) para comprobar si tu dirección de correo electrónico aparece en bases de datos de filtraciones.

Recomendaciones de Seguridad

1

Usar contraseñas únicas

Utiliza contraseñas diferentes para cada servicio. Un gestor de contraseñas puede ayudarte a generarlas y recordarlas.

2

Activar autenticación de dos factores

Añade una capa extra de seguridad a tus cuentas importantes mediante la autenticación de dos factores.

3

Mantener el software actualizado

Actualiza regularmente tu sistema operativo, navegadores y aplicaciones para protegerte contra vulnerabilidades conocidas.
